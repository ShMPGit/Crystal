Инструкция по использованию:
1. Для запуска тестов нужно импортировать проект в IntelliJ IDEA
2. Указать первоначальную информацию (в том случае, если она не была указана) по следующему пути в проекте: src/test/resources
    2.1. Указываемая информация:
        - адрес почтового сервера (например, https://mail.ru/), логин и пароль для входа в почтовый ящик (файл login.properties)
        - ожидаемые тема и содержание входящего письма, а также почтовый адрес отправителя (файл test.xml)
3. Для проверки работоспособности программы использовался ChromeDriver.
В том случае, если текущая операционная система не Linux, а, например, Windows, то в классе WebDriverManager (src/test/java/ru/crystal/driver) добавить соответствующий драйвер браузера под используемую ОС (аналогично для браузера, отличного от Chrome).
4. Запуск автотестов осуществляется путем запуска на выполнение класса MainTest (src/test/java/ru/crystal/tests)
